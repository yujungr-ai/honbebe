# 빠른 시작 가이드

## 한글 인코딩 문제 해결 ✅

FastAPI 응답에 UTF-8 인코딩을 명시적으로 설정했습니다. 백엔드 서버를 재시작하면 한글이 정상적으로 표시됩니다.

## 전체 실행 방법

### 1단계: 백엔드 서버 실행

**터미널 1**에서:
```powershell
# 프로젝트 루트에서
python -m uvicorn app.main:app --host 127.0.0.1 --port 8000
```

### 2단계: 프론트엔드 서버 실행

**터미널 2**에서:
```powershell
# files 디렉토리로 이동
cd files

# 의존성 설치 (최초 1회만)
npm install

# 개발 서버 실행
npm run dev
```

### 3단계: 브라우저에서 접속

브라우저에서 다음 URL을 열어주세요:
```
http://localhost:3000
```

## 웹사이트 기능

웹사이트에서 다음 기능을 사용할 수 있습니다:

1. **검색 폼**: 회사명 또는 종목코드 입력
2. **연도 선택**: 사업연도 선택
3. **보고서 선택**: 사업보고서, 반기보고서, 분기보고서 선택
4. **재무제표 구분**: 연결재무제표 또는 개별재무제표 선택
5. **결과 표시**: 
   - EBITDA 계산 결과
   - 영업이익, 감가상각비, 무형자산상각비 상세 정보
   - 시계열 차트
   - 경고 메시지

## 문제 해결

### 포트 충돌
- 백엔드: 포트 8000이 사용 중이면 `fix_port.ps1` 실행
- 프론트엔드: 포트 3000이 사용 중이면 `npm run dev -- -p 3001`

### 한글 깨짐
- 백엔드 서버를 재시작하세요 (UTF-8 인코딩이 적용됩니다)

### API 연결 오류
- `.env.local` 파일의 `NEXT_PUBLIC_API_URL` 확인
- 백엔드 서버가 실행 중인지 확인

### 컴포넌트 에러
- `npm install`을 다시 실행하세요
- `node_modules` 폴더를 삭제하고 `npm install`을 다시 실행하세요
